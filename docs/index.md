# MC Vanilla+ Modpack

Welcome to the official documentation for the MC Vanilla+ modpack.

This pack focuses on:
- Vanilla-feeling gameplay
- Strong performance
- Carefully curated visual and QoL mods

---

## Getting Started

### Requirements
- Minecraft 1.21.1
- Fabric Loader
- Java 21
- Modrinth App (recommended)

### Installation
1. Download the `.mrpack` file
2. Open the Modrinth App
3. Import the modpack
4. Launch and play
